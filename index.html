RESET CSS

What is Reset CSS

Reset CSS is a technique in web development used to remove default CSS values for HTML elements. The goal of Reset CSS is to make HTML elements look the same across different browsers by removing any differences in default display styles set by browsers.

Why Reset CSS

Different browsers have different default styles for HTML elements. For example, the default margin and padding for the body element in Chrome is 8px, while in Firefox it is 16px. This can cause a lot of problems when you want to make your website look the same across different browsers. Reset CSS is used to remove these differences and make your website look the same across different browsers.
